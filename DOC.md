## Folder Structure

After creation, your project should look like this:

```
my-app/
  README.md
  node_modules/
  package.json
  public/
    index.html
    favicon.ico
  src/
    api/
        userApi.js
        productApi.js
    components/
        Button.js
        Icon.js
        UserDetail.js
        UserList.js
    containers/
        App.css
        App.js
        App.test.js
        HomePage.js
        UserDetailPage.js
        UserListPage.js
        index.css
    images/
        logo.svg
    index.js
    utils/
        testUtils.js
```
